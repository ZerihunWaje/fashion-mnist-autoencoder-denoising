# Fashion MNIST Autoencoder Denoising

This project demonstrates the use of an autoencoder neural network to remove noise from Fashion MNIST images. It's a practical example of how unsupervised learning can be used for image denoising tasks.

---

## Project Goals

- Train a convolutional autoencoder on Fashion MNIST.
- Add synthetic noise to the dataset.
- Reconstruct the original image from the noisy input.
- Visualize and evaluate denoising performance.
